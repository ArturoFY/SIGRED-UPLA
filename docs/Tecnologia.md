# Stack Tecnológico

Aquí detallamos las herramientas utilizadas para el desarrollo, control de calidad y despliegue del proyecto SIGRED-UPLA.

* **Frontend:** React + Tailwind CSS (para la interfaz móvil y dashboard).
* **Backend:** Laravel 11 (PHP).
* **Base de Datos:** MySQL.
* **Control de Versiones:** Git & GitHub.
* **CI/CD:** GitHub Actions (automatización de despliegue).
* **Análisis de Calidad:** SonarCloud (monitoreo de deuda técnica y calidad de código).
* **Monitoreo:** Sentry (seguimiento de errores en tiempo real en producción).
